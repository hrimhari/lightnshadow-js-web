O(n2) Light and Shadow renderer in HTML5

This is a conversion from a Computer Graphics exercise I once did as a Java applet. A while ago it became virtually impossible to run Java applets on a browser, so I decided to create this Javascript + HTML5 version.

Choose a height map from your device, then experiment with the buttons below.

Here are some suggestions of online maps:

  • carassonet.org/lightnshadow/hills.gif
  • playerstage.sourceforge.net/doc/Gazebo-manual-svn-html/terrain.png
  • www.videotutorialsrock.com/opengl_tutorial/terrain/heightmap.png
